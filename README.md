# WooCommerce Quick Order Search

Quick admin-side WooCommerce order search by order ID or customer phone number.

WooCommerce Quick Order Search adds a floating search button to the WordPress admin area, allowing WooCommerce managers to quickly find orders by entering an order ID or customer phone number. It supports Persian, Arabic, and English digits, normalizes phone numbers, and displays useful order details in a clean modal interface.

## Features

- Quick order search inside the WordPress admin area
- Search by WooCommerce order ID
- Search by customer billing phone number
- Support for Persian, Arabic, and English digits
- Automatic phone number normalization
- Compatible with WooCommerce HPOS
- Clean modal-based admin interface
- Displays order status, customer details, total amount, payment method, and order date
- Quick access to order edit page
- Customer profile and customer orders links when available
- Copy buttons for order number, phone number, and email
- Lightweight inline assets
- Access limited to users with WooCommerce management permission

## Requirements

- WordPress
- WooCommerce
- PHP 7.4+

# WooCommerce Quick Order Search

جستجوی سریع سفارشات ووکامرس در محیط مدیریت وردپرس.

افزونه WooCommerce Quick Order Search یک دکمه شناور به محیط مدیریت وردپرس اضافه می‌کند تا مدیران ووکامرس بتوانند سفارش‌ها را به‌سرعت با شماره سفارش یا شماره موبایل مشتری پیدا کنند. این افزونه از اعداد فارسی، عربی و انگلیسی پشتیبانی می‌کند، شماره موبایل را نرمال‌سازی می‌کند و اطلاعات مهم سفارش را در یک پنجره تمیز و کاربردی نمایش می‌دهد.

## امکانات

- جستجوی سریع سفارش‌ها در محیط مدیریت وردپرس
- جستجو بر اساس شماره سفارش ووکامرس
- جستجو بر اساس شماره موبایل مشتری
- پشتیبانی از اعداد فارسی، عربی و انگلیسی
- نرمال‌سازی خودکار شماره موبایل
- سازگار با ساختار HPOS ووکامرس
- رابط کاربری تمیز و مودال در پنل مدیریت
- نمایش وضعیت سفارش، اطلاعات مشتری، مبلغ کل، روش پرداخت و تاریخ سفارش
- دسترسی سریع به صفحه ویرایش سفارش
- نمایش لینک پروفایل کاربر و سفارشات مشتری در صورت وجود
- دکمه کپی برای شماره سفارش، شماره موبایل و ایمیل
- استفاده از CSS و JS سبک به‌صورت inline
- محدود بودن دسترسی به کاربران دارای مجوز مدیریت ووکامرس

## پیش‌نیازها

- ووکامرس
- PHP 7.4 یا بالاتر
